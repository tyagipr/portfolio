
# Unique Portfolio Website

This is a code bundle for Unique Portfolio Website. The original project is available at https://www.figma.com/design/Kspke041VBXC2jaXQh8Bcq/Unique-Portfolio-Website.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions. The workflow builds the React app and deploys it to `https://tyagipr.github.io`.
